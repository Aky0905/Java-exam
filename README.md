1. 개요
목적: 시간표를 저장하고 불러오면서 작성할 수 있는 프로그램이 필요할 거 같아서 만들었습니다.

대상: 청주대 재학생들입니다.

2.1 중요성:
이 프로그램을 통해 언제 몇시에 수업이 있는지 확인할 수 있습니다.

2.2 필요성:
학교를 다니면서 학교엔 시간표 관련 프로그램이 없다 생각해서 만들었습니다.

4.느낌점:
실제 문제를 해결하기 위해 다양한 접근 방식을 시도하며 문제 해결 능력이 크게 향상되었습니다.
